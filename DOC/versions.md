官方说：“DevEco Studio 3.1配套支持HarmonyOS 3.1及以下版本的应用及服务开发”

但经过测试，使用Stage模型的APP，是可以构建并且安装到HarmonyOS 4.2的Mate60Pro设备上 ，但是不能安装到HarmonyOS 2.0上。
